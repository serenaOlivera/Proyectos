## Proyecto ticket-API

El proyecto se basa en entender el framework spring boot, a través de una API que genera,lista, y marca como pagado tickets.

### Tecnologías
Trabajamos con:
- Spring Boot v4.1.0
    
    Generar un spring intializer con los siguientes valores:
    - Project: Maven
    - Language: Java
    - Spring Boot: versión estable disponible 
    - Group: com.example 
    - Artifact: ticket-api
    - Packaging:Jar
    - Java: 21
    - Dependencias: Spring web. 

- Apache Maven 3.8.7

    ``sudo apt install maven``
- Java 21
